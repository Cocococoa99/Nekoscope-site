NekoScope Privacy Policy

Last updated: 2026-05-28

1. Overview

NekoScope is a read-only creator analytics tool for reviewing TikTok video performance, organizing content records, and generating creator reports. This Privacy Policy explains what data NekoScope may access, how it is used, where it is stored, and how users can manage or delete it.

2. Data NekoScope May Access

After the user authorizes NekoScope through TikTok Login Kit, NekoScope may access data allowed by the approved TikTok API scopes selected by the user and approved for the app. This may include:

  • Basic account information, such as open_id, display name, and avatar URLs through user.info.basic.
  • Profile information, such as bio description and profile link through user.info.profile.
  • Account-level statistics, such as follower count, following count, likes count, and video count through user.info.stats.
  • Public video information, such as public video IDs, titles or descriptions, publish time, duration, public URLs, views, likes, comments, and shares through video.list.

3. OAuth Tokens and Credentials

NekoScope may store OAuth access tokens and refresh tokens locally so that the user can continue using authorized API features. NekoScope does not publish these tokens publicly. Users should keep API credentials, client secrets, access tokens, refresh tokens, and local database files private and should not upload them to public repositories.

4. How Data Is Used

NekoScope uses authorized data only to provide creator analytics features, including video performance review, public video list import, API reconciliation, local metrics snapshots, content organization, and report generation.

5. Data Storage

NekoScope stores imported data locally on the user's device or in a user-controlled database or export file. NekoScope does not require a centralized cloud database for its basic local analytics workflow.

6. Data Sharing and Selling

NekoScope does not sell, rent, or trade user data. NekoScope does not share user data with third parties except when the user chooses to export, back up, upload, or otherwise share their own files.

7. No Posting or Automated Account Actions

NekoScope does not publish videos, upload content, like, comment, follow, send messages, or perform automated account actions. It is a read-only analytics tool.

8. Data Retention and Deletion

Users control locally stored NekoScope data. Users may delete their local database, backups, exported files, or imported records at any time. Users may also revoke TikTok authorization through TikTok or clear locally stored OAuth tokens from NekoScope.

9. Security

NekoScope uses local storage and masks sensitive token values in the user interface where practical. However, no storage method is completely secure. Users are responsible for protecting their device, local files, API credentials, client secrets, access tokens, and refresh tokens.

10. Children's Privacy

NekoScope is intended for use by creators and account owners who are authorized to access their own TikTok data. NekoScope is not intended to collect data from children.

11. Changes to This Privacy Policy

This Privacy Policy may be updated from time to time. The updated version will be posted on this page with a revised "Last updated" date.
