NekoScope

NekoScope is a read-only creator analytics tool for reviewing TikTok video performance, organizing content records, and generating creator reports.

What NekoScope Does

NekoScope helps authorized users import and organize their own TikTok account information, profile information, account-level statistics, and public video metadata using TikTok-approved read-only API scopes.

The imported data can be used for content review, performance tracking, API reconciliation, and local creator analytics reports.

Read-Only Use

NekoScope does not publish videos, upload content, like, comment, follow, send messages, or perform automated account actions. It only reads data that the user has authorized through TikTok Login Kit.

TikTok Integration

NekoScope uses TikTok Login Kit for user authorization. After the user authorizes access, NekoScope uses approved read-only scopes such as user.info.basic, user.info.profile, user.info.stats, and video.list to display authorized account information and public video data.
