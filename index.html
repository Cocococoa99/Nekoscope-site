How to use NekoScope with TikTok

NekoScope is designed for TikTok creators who want to review their own authorized public video data in a local analytics workspace. The website provides the public product entry point, while the actual analytics dashboard runs locally on the creator's device.

Before starting

  • Open the local NekoScope app on your own device.
  • Select or create a creator account profile inside NekoScope.
  • Use the TikTok Login entry point to authorize read-only access.
  • Review the requested scopes before granting authorization.

What NekoScope imports

  • Authorized TikTok creator profile information.
  • Public video metadata, such as title, publish time, duration, and public URL.
  • Public video engagement counts, such as views, likes, comments, and shares.
  • Reconciliation records that help avoid duplicate imports.

1. Connect TikTok

The creator starts TikTok Login Kit from NekoScope and authorizes read-only access through TikTok's official consent screen.

2. Preview API data

NekoScope fetches authorized user information and public video list data, then displays the results for review before saving anything.

3. Reconcile videos

The app compares TikTok API results with the local database and marks each item as a new video, changed metric snapshot, metadata update, or duplicate.

4. Import snapshots

After confirmation, NekoScope stores selected video records and metric snapshots in the creator's local database.

5. Review performance

The creator can review video performance, compare snapshots, add manual notes, and use labels to support future content decisions.

6. Stay in control

OAuth tokens and analytics records are handled in the local NekoScope app. This static website does not store TikTok tokens or analytics data.

Read-only boundary: NekoScope does not upload, publish, edit, delete, like, comment, follow, message, or automate TikTok account actions.

How users complete TikTok login

1. Open NekoScope locally

The user runs the NekoScope Streamlit app on their own device and opens the TikTok API page.

2. Start TikTok Login Kit

The user clicks the authorization entry point to open TikTok's official login and consent screen.

3. Authorize read-only scopes

The user grants read-only scopes such as user profile, account stats, and public video list access.

4. Return through callback

TikTok redirects back to the registered callback URL. The authorization code is used by the local app to complete token exchange.

5. Import public video data

NekoScope reads authorized public video metadata and metrics, then shows a reconciliation preview before importing.

6. Analyze locally

The creator reviews snapshots, trends, and performance labels in a local database under their control.

Requested TikTok scopes

user.info.basicUsed to identify the authorized creator account. user.info.profileUsed to display basic profile information inside the local analytics dashboard. user.info.statsUsed to display authorized account-level creator statistics. video.listUsed to read the creator's own public video list and public video engagement counts.

No Content Posting API

NekoScope does not use the Content Posting API. It does not upload, publish, schedule, edit, delete, like, comment, follow, send messages, scrape other users, or perform any automated account actions.

The app is designed only for the authorized creator to analyze their own TikTok content data.
