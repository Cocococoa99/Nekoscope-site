NekoScope Privacy Policy

Last updated: 2026-05-27

1. Overview

NekoScope is a private creator analytics tool for reviewing TikTok video performance, organizing video metadata, and generating content review reports.

2. Data We May Access

After user authorization, NekoScope may access basic TikTok account information and public video information, such as video IDs, titles, publish times, durations, and public performance metrics available through approved API permissions.

3. Data Storage

NekoScope stores data locally on the user's device or in a user-controlled database. The data is used only for video review, analytics, and content organization.

4. Data Sharing

NekoScope does not sell, rent, transfer, or share user data with third parties.

5. No Automated Account Actions

NekoScope does not post content, like, comment, follow, send messages, or perform automated actions on behalf of users.

6. Data Deletion

Users may delete their local database, backups, exported files, or disconnect API authorization at any time.

7. Security

Users should keep API credentials, client secrets, access tokens, and refresh tokens private and should not publish them in public repositories.
